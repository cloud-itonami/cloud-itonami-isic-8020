# Operator Guide

## First Deployment
1. Register operator, monitoring center/service area, licensing and
   response-time scope, technicians and robots.
2. Import existing client installation and monitoring history.
3. Run read-only licensing-scope and installation-support robot
   mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run follow-up record and audit export.

## Minimum Production Controls
- licensing-scope and response-time validation before any dispatch
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (an out-of-scope
  alarm response, an unpermitted installation job, an unverified
  monitoring-center escalation)
- evidence-backed follow-up records
- audit export for every dispatch, sign-off and follow-up record
- backup manual security-systems-service process

## Certification
Certified operators must prove robot-safety integrity, licensing/
response-time-scope discipline, evidence-backed follow-up records and
human review for dispatch-affecting actions.
